# HandyMatches - Neighbors Helping Neighbors

A comprehensive volunteer-resident matching platform connecting local volunteers with community members who need assistance with everyday tasks.

## Features

- **Volunteer Management**: Sign up, background checks, availability scheduling
- **Resident Requests**: Multi-step help request forms
- **Admin Dashboard**: User management, communications, analytics, team chat
- **Community Gallery**: Showcase completed projects
- **Responsive Design**: Works on desktop and mobile

## Tech Stack

- React 18
- Tailwind CSS
- Font Awesome icons
- Pure client-side application (no backend required)

## Deployment

This is a single-page application that can be deployed to any static hosting service.

### Deploy to Netlify

1. Fork this repository
2. Connect your GitHub account to Netlify
3. Select this repository
4. Deploy settings:
   - Build command: (leave empty)
   - Publish directory: `.`
5. Click "Deploy site"

## Local Development

Simply open `handymatches-fixed.html` in your browser. No build process required.

## Admin Access

Click the "Admin" link in the navigation to access the admin dashboard with:
- User management and approval
- Background check tracking
- Platform analytics
- Team communications
